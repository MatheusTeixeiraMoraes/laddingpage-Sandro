# Sobre Mim / Autoridade Pessoal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Página `/sobre` com bio, galeria de fotos, depoimentos em vídeo, avaliações do Google (mock) e redes sociais — acessível via link no header. Todo conteúdo é placeholder claramente isolado em arquivos de dados, pronto pra troca por conteúdo real depois.

**Architecture:** Página inteiramente server component (sem estado, sem interatividade) montada a partir de 5 componentes de seção, cada um lendo de um arquivo de dados mock. Nenhuma lógica condicional não-trivial nesta fatia, então sem teste automatizado novo (consistente com o restante do projeto: só lógica com branching real ganha `node --test`).

**Tech Stack:** Next.js 16 (App Router) + TypeScript + Tailwind CSS v4.

## Global Constraints

- Todo texto/imagem/vídeo/link é placeholder — nenhum conteúdo real foi fornecido ainda.
- Sem integração real com Google Places API — `GoogleReview` é só um tipo mock pensado pra já ter o formato certo quando a API entrar.
- Sem ícones de terceiros — links de rede social usam o mesmo estilo de pill/botão já usado no resto do site (`rounded-full`, borda, hover), não SVG de logo de marca.
- Nenhum novo teste — esta fatia não tem lógica de filtro/cálculo como os pilares anteriores.

function estrelas(nota: number): string {
  return "★".repeat(nota) + "☆".repeat(5 - nota);
}

export function GoogleReviews() {
  return (
    <section className="flex w-full flex-col items-center gap-4">
      <h2 className="text-2xl font-bold text-brand-purple">Avaliações no Google</h2>
      <div className="grid w-full grid-cols-1 gap-4 sm:grid-cols-2">
        {googleReviews.map((review) => (

            key={review.id}
            className="flex flex-col gap-1 rounded-2xl border border-slate-100 p-4 text-left shadow-sm"
